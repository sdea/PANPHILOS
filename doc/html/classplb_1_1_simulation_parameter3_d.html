Detailed Description

template<typename T>
class plb::SimulationParameter3D< T >

Class which incorporates the simulation parameters.

The parameters are read by the constructor from an XML file in input. The constructor also assign dependend parameters such as dt, and converts the contact angle from decimal to radiant
